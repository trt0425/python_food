# petfeeder

修習資料結構的Final Project： 自動餵食器

開發環境：Raspberry Pi + python3.6

利用python GPIO 控制步進馬達去做定時旋轉,若想手動控制也可以使用按鈕去觸發。

飼料擋板使用3D列印做出螺旋柱狀體，使得推動飼料及防止過多倒出。
